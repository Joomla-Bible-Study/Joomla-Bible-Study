Series Help Screen

Click on the New link to add a new Series. Click a link to edit a series, or the checkbox followed by Edit on the top menu.

Put in a name for the series and click Save. You can edit them. To delete rows you need to set the Allow Admin Delete Rows option under Preferences. But be careful. If you delete a series that is attached to a study it may cause unintended results.
